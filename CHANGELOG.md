# Changelog

All notable changes to this project are documented in this file.

## 1.0.16 - 2026-03-23

- Extended README documentation for `size` support (`number | string`) with CSS value examples.
- Added a `createSvgIcon` factory example that shows how to build project-specific icon wrappers.

## 1.0.15 - 2026-03-23

- Added support for CSS-based icon sizing by allowing `size` to be `number | string`.
- Updated `setDefaultIconSize` to accept both numeric and string values.
